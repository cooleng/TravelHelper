# TravelHelper
毕设-出行助手 APP（Android） 

技术栈：rxjava + retrofit + butterKnife + netty
<br/>
功能：  搜周边、路线规划、备忘录、通讯录、朋友圈发帖、聊天等
<br/>
效果：

<div>
(1)搜周边：选择一个地点，调用百度地图API获取POI信息，并渲染。

</div>
<br/>
<div>
(2)调用百度地图SDK 与API，对目的地进行导航或路线规划。
<img src="./images/%E5%AF%BC%E8%88%AA.png"  height="752" width="424">
</div>
<br/>
<div>
(3)一个富文本编辑的备忘录功能（UI参考MIUI）
<img src="./images/%E5%A4%87%E5%BF%98%E5%BD%95%E5%88%97%E8%A1%A8.png"  height="752" width="424">
</div>
<br/>
<div>
(4)首字母排序通讯录

</div>
